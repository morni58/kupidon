# Дизайн-промпт для CupidBot (Telegram Mini App, дейтинг)

> Скопируй ВЕСЬ текст ниже и отправь в дизайн-генератор. Он рассчитан на выдачу мобильных экранов в HTML + Tailwind, которые потом 1-в-1 переносятся в React + Tailwind + Framer Motion.

---

Ты — продуктовый дизайнер мирового уровня (уровень Tinder, Bumble, Hinge, Gen-Z приложений). Сделай законченный UI-кит и все экраны мобильного дейтинг-приложения **CupidBot** — это **Telegram Mini App** (открывается внутри Telegram на телефоне). Дизайн должен быть «вкусным», современным, дорогим и эмоциональным — не корпоративным, не сухим, не «как госуслуги».

## 0. Формат вывода
- Отдай каждый экран как отдельный самодостаточный блок на **HTML + Tailwind CSS** (через `cdn.tailwindcss.com`), шрифт **Onest** (Google Fonts), иконки **Phosphor** (`@phosphor-icons/web`).
- Каждый экран — в рамке телефона **390×844** (iPhone), `border-radius: 3rem`, контент занимает 100% высоты, без вертикального скролла на главных экранах (кроме Профиля и Чатов — там скролл допустим).
- Используй ТОЛЬКО реальные значения из дизайн-системы ниже. Никаких новых цветов/шрифтов.
- Покажи каждый экран в трёх состояниях темы там, где это уместно: **Light**, **18+**, **Oligarch**.
- Для фото людей используй красивые плейсхолдеры (можно `https://placehold.co/...` или абстрактные градиенты), но размести их продуманно — как в премиальном дейтинге.

## 1. Дизайн-система (строго)
**Палитра:**
- Фуксия (основной бренд): `#FF00FF`
- Розовый (вторичный): `#FF66CC`
- Градиент бренда: `linear-gradient(135deg, #FF00FF 0%, #FF66CC 100%)`
- Тёмный текст/фон: `#0F0F13`
- Светлый фон: `#FAFAFC` с паттерном из точек `radial-gradient(circle, #e5e7eb 1px, transparent 1px); background-size: 24px 24px`
- Золото (VIP/Олигарх): `#FFD700`, градиент VIP `linear-gradient(135deg, #FFE259 0%, #FFA751 100%)`
- Изумруд (успех/верификация-щит): `#10B981`
- 18+ режим: фон `#0A0000`, акцент `#FF3333`, рамки `#4A0000`, градиент `linear-gradient(135deg,#4A0000,#1A0000)`
- Синий (Verified-галочка): `#3B82F6`

**Стиль:**
- Шрифт **Onest**, веса 300–900. Заголовки — 800/900 (font-black), текст — 500/600.
- **Glassmorphism**: панели `background: rgba(255,255,255,0.85); backdrop-filter: blur(12px); border: 1px solid rgba(255,255,255,0.5)`.
- Скругления крупные: `rounded-[2rem]` для карточек/модалок, `rounded-[1.5rem]` для блоков, `rounded-full` для кнопок-действий и аватарок.
- Тени мягкие, цветные: `box-shadow: 0 30px 60px -15px rgba(255,0,255,0.2)`.
- Свечение для заголовков: `text-shadow: 0 0 20px rgba(255,0,255,0.5)`.
- Анимации: spring-физика (как Framer Motion), 60fps. Вместо спиннеров — скелетоны.
- Учитывай **safe-area** (вырез/«чёлка» сверху, домашняя полоса снизу).
- Всё рассчитано на тап пальцем: кнопки-действия минимум 48–56px.

**3 темы (через CSS-переменные):**
- **Light** — по умолчанию, светлая, фуксия.
- **18+** — тёмно-бордовая, акцент красный `#FF3333`, ощущение «приватной комнаты».
- **Oligarch** — чёрная с золотом `#FFD700`, премиальная, «дорогая интрига».

## 2. Экраны (сделай ВСЕ)

### A. Онбординг / Регистрация — ПЕРЕДЕЛАТЬ из «анкеты госоргана» в игривый флоу
Главная боль текущего дизайна: регистрация сухая и формальная. Сделай её **лёгкой, дружелюбной, по одному вопросу на экран**, с крупным заголовком-вопросом, прогресс-баром сверху (тонкая градиентная полоса), большими тапабельными карточками-вариантами и плавными переходами. Минимум полей, максимум эмоций и микрокопирайта.

Шаги (каждый — отдельный экран):
1. **Имя** — крупное приветствие «Как тебя зовут?», одно поле, мягкая подсказка. Без фамилии.
2. **Возраст** — ВАЖНО: НЕ календарь и НЕ дата рождения. Сделай **красивый горизонтальный пикер/слайдер возраста (18–60) или крупный barrel-picker с числом по центру**. Пользователь выбирает просто число лет. Внизу мелким — «Тебе должно быть 18+».
3. **Я** — две большие карточки [Парень] [Девушка] с иконками/эмодзи, выбор тапом, без кнопки «далее».
4. **Ищу** — [Парня] [Девушку] [Всех].
5. **Геолокация** — большая кнопка «📍 Найти меня» + поле поиска города с автокомплитом (выпадающий список). Можно «Пропустить».
6. **Фото** — сетка 5 слотов 3×4, первый слот крупнее и помечен «Главное». Загрузка тапом, превью, крестик для удаления, индикатор загрузки. Сделай это **самым красивым шагом** — фото это сердце дейтинга.
7. **Интересы/теги** — облако кнопок-пилюль с эмодзи и цветной обводкой по HEX тега; счётчик «выбрано N/5»; при 5 выбранных остальные мягко блокируются.
8. **О себе** — textarea (до 150 симв.) со счётчиком, необязательно, дружелюбная подсказка-пример.
9. **Финал** — «Готово! 🚀» с лёгким конфетти и кнопкой «В ленту».

### B. Лента свайпов (главный экран) — в стиле Tinder/Bumble
- Карточка-фото на ВЕСЬ экран, скруглённая `rounded-[2rem]`.
- Сверху на фото — индикаторы-палочки текущего фото (как в Stories), тап по левой/правой половине листает фото внутри анкеты.
- Бейдж «✓ Verified» (синий, glass) в углу.
- Снизу — градиент в чёрный, поверх: **Имя, возраст** крупно, под ним «📍 Город · N км», ряд тегов-пилюль (стекло), бейдж «✓ N общих тегов» (изумрудный).
- При свайпе вправо проявляется штамп «ЛАЙК ❤️» (зелёный, повёрнутый), влево — «НЕТ 👎» (красный), вверх — «СУПЕРЛАЙК ⭐».
- Нижняя панель из 5 круглых кнопок: ⏪ Rewind (маленькая, amber), ✕ (большая, белая), ♥ (большая, градиент-бренд, со свечением), ⭐ Суперлайк (маленькая, фиолетовая, с бейджем-счётчиком).
- Покажи отдельно состояние **Пейволл** (свайпы кончились): фон-карточка блюрится, поверх — сочная модалка с таймером «до 00:00 МСК» и переливающейся кнопкой «Стать Купидоном 👑».
- Покажи модалку **МЭТЧ**: полноэкранная, конфетти, два аватара, «💕 Совпадение!», кнопки «Написать первым» и «Продолжить».

### C. Профиль — ПЕРЕДЕЛАТЬ: сделать объёмным, листаемым, премиальным (главная просьба)
Сейчас профиль «скучный/яичный». Нужен **богатый, прокручиваемый профиль-витрина**, как в топовых дейтингах:
- **Верх**: большое главное фото-обложка на всю ширину (или карусель фото с индикаторами), поверх снизу — имя, возраст, ✓Verified, город.
- **Карусель фото**: листается свайпом, точки-индикаторы, тап открывает фото на весь экран.
- **Шкала «Горячести» (Profile Score 0–100)** — красивый прогресс-бар с градиентом и подсказкой «добавь видео и пройди верификацию для 100%».
- **Стрик** — 🔥 «N дней подряд» с мягкой подсветкой.
- **Блок статистики** — 3 карточки (свайпы / суперлайки / «кто смотрел тебя») в стекле.
- **Теги-интересы** — пилюли как на карточке.
- **«О себе»** — аккуратный блок с текстом.
- **Переключатели** (красивые тумблеры): 🔞 Комната 18+ (с подписью «только verified»), 🛡 Анти-Олигарх щит (для девушек — «бесплатно навсегда»), и для Kupidon — отдельная **золотая Oligarch-панель** (стелс-режим + счётчик VIP-сигналов «6/20»).
- **CTA апгрейда** для Free — яркая градиентная карточка «Стань Купидоном 💘».
- Всё лаконично, с воздухом, сгруппировано по карточкам-секциям. Не нагромождать.

### D. Симпатии (кто лайкнул)
- Лента карточек тех, кто лайкнул. Фото ОТКРЫТЫ у всех (без блюра — принципиально).
- Обычная карточка: фото, имя+возраст, «лайкнул(а) недавно», кнопка «Написать».
- **VIP-карточка Олигарха**: золотая рамка, размытый силуэт с золотым 👑, текст «👑 VIP проявил интерес», анонимная записка, кнопка «Раскрыть» (золотая). Всплывает первой.
- Секция «Кто смотрел тебя»: у Free — силуэты + счётчик (триггер апгрейда), у Premium — открыто.

### E. Список чатов
- Карточки матчей: аватар (или 💕/🔥 для 18+), имя, превью последнего сообщения, время, бейдж-точка непрочитанного. 18+ чаты — с красной полоской слева.

### F. Диалог (чат)
- Шапка: аватар, имя + ✓, статус «в сети», кнопка ⚠️ (репорт), кнопка ✈️ Telegram (серая/disabled с подписью «ещё N смс», синеет при разблокировке).
- Пузыри: свои — градиент-бренд, скруглены `rounded-tr-none`; чужие — серые `rounded-tl-none`.
- **ConsentBubble** (системный): голубая карточка по центру «Собеседник хочет в Telegram» + [Отклонить] [Одобрить].
- **Ледоколы**: над клавиатурой 3 кнопки-подсказки (стекло, фуксия) когда сообщений нет.
- Поле ввода: круглое, скрепка, кнопка отправки (градиент). В 18+ чате скрепка меняется на 🔥 (исчезающие фото).

### G. Верификация
- Экран селфи-liveness: круг с пунктиром, иконка лица, бейдж со случайным жестом («Коснись щеки 👆»), заголовок «Подтверди, что это ты», подзаголовок «Кадры не сохраняются», кнопка «📷 Начать», обещание «✓ Синяя галочка +15 к анкете». Состояния: интро / запись (пульс) / успех (✅ конфетти) / ошибка.

### H. Нижний таб-бар (на всех главных экранах)
- 4 иконки: 🃏 Лента · ❤️ Симпатии · 💬 Чаты · 👤 Профиль. Активная — подсвечена фуксией и чуть увеличена. Бейдж-точка на новых событиях. Стекло + blur.

### I. Спец-режимы (показать темы)
- **18+ комната**: та же лента/чат, но тёмно-бордовая тема, красные акценты, плашка «Эротика заблокирована системой», теги намерений (🎲 Без обязательств, 🌙 Один вечер).
- **Oligarch**: чёрно-золотая панель управления, тумблер «Стелс-режим», счётчик сигналов, карточка «после взаимного лайка — раскрыт».

## 3. Жёсткие требования к качеству (без ошибок)
- **Никаких логических ошибок**: возраст — число (не дата); кнопки ведут в правильные экраны; счётчики/лимиты консистентны (Free 50 свайпов, Premium 200, Kupidon 500); состояния пустых экранов проработаны («Анкеты закончились», «Нет чатов»).
- **Никаких дизайнерских ошибок**: единые отступы (используй шкалу 4/8/12/16/24), выравнивание по сетке, единые скругления и тени, контраст текста ≥ AA, иконки одного семейства (Phosphor), консистентные кнопки. Тексты на русском, живые и тёплые, без канцелярита.
- Фото и аватары — всегда `object-cover`, без растяжений; главное фото обязательно; пустые слоты — красивые плейсхолдеры, а не серые квадраты.
- Адаптив под маленькие экраны (от 360px), ничего не обрезается, ничего не наезжает.
