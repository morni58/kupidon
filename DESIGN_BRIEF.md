# ЧАСТЬ 0. ЧТО ЭТО ЗА ПРОДУКТ (чтобы ты понимал смысл каждого элемента)

**CupidBot** — приложение для знакомств, которое открывается ВНУТРИ мессенджера Telegram как «Mini App» (веб-приложение в окне Telegram на телефоне). Пользователь нажимает кнопку в боте — открывается наше приложение на весь экран телефона.

Суть продукта: люди регистрируются, загружают фото, листают анкеты других людей (свайп вправо = «нравишься», влево = «нет»). Если двое свайпнули друг друга вправо — возникает «мэтч» (совпадение), открывается чат. Есть платные функции, спец-режимы (18+ комната, режим «Олигарх»), верификация по селфи, защита от ботов.

Аудитория: молодёжь и взрослые 18–40, СНГ, русскоязычные. Тон — тёплый, игривый, современный, чуть дерзкий. НЕ официальный, НЕ корпоративный, НЕ «госуслуги». Эмоция первого впечатления: «вау, красиво и хочется пользоваться».

**Три типа подписки (это влияет на то, что видит пользователь):**
- **Free** (бесплатно): 50 свайпов в день, 0 суперлайков, написать без мэтча — только за деньги (Telegram Stars), Telegram-контакт открывается после 15 сообщений, «кто меня смотрел» — только силуэты.
- **Premium**: 200 свайпов/день, 5 суперлайков/день, 3 «врыва» в чат/день, контакт после 5 сообщений, есть «откат» свайпа (Rewind), «кто смотрел» — открыто.
- **Kupidon (VIP)**: 500 свайпов/день, 5+ суперлайков, 15 «врывов»/день, контакт сразу, режим «Олигарх», абсолютный приоритет в ленте.

**Платные разовые покупки за Telegram Stars (⭐ — внутренняя валюта Telegram):** Написать без мэтча — 50⭐; Буст анкеты на 2 часа — 100⭐; Разовый суперлайк — 150⭐; Анонимный VIP-сигнал (Олигарх) — 500⭐.

**Спец-механики (нужны отдельные экраны/состояния):**
- **Верификация**: пользователь делает селфи с случайным жестом → получает синюю галочку «Verified». Поднимает доверие.
- **Комната 18+**: отдельный изолированный режим для взрослых с открытыми намерениями (не порно — эротика ЗАПРЕЩЕНА). Вход только для verified. Тема приложения становится тёмно-бордовой. Фото в чате «исчезающие» (посмотрел — сгорело).
- **Режим «Олигарх»** (только Kupidon): анкета становится невидимой в общей ленте («стелс»), но он шлёт девушкам анонимный приоритетный VIP-сигнал. Девушка видит золотую карточку «👑 VIP проявил интерес» с размытым силуэтом и решает — «раскрыть» (= лайк) или нет. Имя/фото открываются только при взаимности. Лимит 20 сигналов/день.
- **Анти-Олигарх щит** (бесплатно для девушек): один тумблер — делает девушку полностью невидимой для всех «Олигархов».
- **Дейли-стрик**: заходишь каждый день → растёт счётчик 🔥, на 3/7/14 день — награды.
- **«Кто смотрел тебя»**: список тех, кто заходил в твою анкету.

# ЧАСТЬ 1. ТЕХНИЧЕСКИЙ КОНТЕКСТ (обязательно учесть)

- Это **мобильный веб-экран внутри Telegram**. Рисуй всё в рамке телефона **390 × 844 px** (iPhone 14), скругление корпуса `border-radius: 3rem`, рамка `8px solid #111`, сверху имитация «чёлки».
- Контент занимает **100% высоты экрана**. На главных экранах (Лента, Онбординг-шаги, Верификация) **вертикального скролла быть НЕ должно** — всё помещается. На Профиле, Списке чатов, Диалоге, Симпатиях — вертикальный скролл разрешён.
- Учитывай **safe-area**: сверху отступ под «чёлку» (~44px), снизу под «домашнюю полосу» (~34px).
- Все интерактивные элементы — под палец: кнопки-действия минимум **48–56px**, теги/пилюли минимум 32px высотой.
- Внизу почти всех экранов — **таб-бар** (панель навигации, описана в Части 4).
- Используй HTML + Tailwind CSS (через `cdn.tailwindcss.com`), шрифт **Onest** (Google Fonts, веса 300–900), иконки **Phosphor Icons** (`@phosphor-icons/web`, классы вида `ph-fill ph-heart`, `ph-bold ph-x`).

## 2.2 Типографика (шрифт Onest)
- H1 (заголовок-вопрос онбординга, имя в профиле): 32–40px, вес 900 (font-black), tracking-tight.
- H2 (заголовки секций): 24px, вес 800.
- H3 (подзаголовки): 18–20px, вес 700.
- Body (основной): 15–16px, вес 500.
- Caption (подписи, мелкое): 11–13px, вес 600, иногда uppercase + tracking-wider.
- Имя+возраст на карточке свайпа: 24–28px вес 700, белым поверх фото.

## 2.3 Спейсинг и сетка
- Шкала отступов строго: **4 / 8 / 12 / 16 / 24 / 32px**. Никаких произвольных значений.
- Боковые поля экрана: 16–20px.
- Расстояние между карточками-секциями: 12–16px.

## 2.4 Радиусы скругления
- Карточки/модалки: `2rem` (32px).
- Блоки/инпуты: `1.5rem` (24px) или `1rem` (16px).
- Кнопки-действия (круглые) и аватарки: `9999px` (полный круг).
- Пилюли-теги: `9999px`.

## 2.5 Тени и эффекты
- Карточка-фото: `box-shadow: 0 30px 60px -15px rgba(255,0,255,0.2)`.
- Glassmorphism (стеклянные панели): `background: rgba(255,255,255,0.85); backdrop-filter: blur(12px); border: 1px solid rgba(255,255,255,0.5)`.
- Кнопка лайка: свечение `box-shadow: 0 0 0 + цветная тень rgba(255,0,255,0.5)`.
- Свечение заголовков (по желанию): `text-shadow: 0 0 20px rgba(255,0,255,0.5)`.

## 2.6 Состояния компонентов (описать визуально для всех)
**Кнопки:**
- Primary: фон — градиент бренда, текст белый, вес 700, высота 48–56px, радиус 16px. Состояния: обычная / нажата (scale 0.98) / disabled (opacity 0.4).
- Secondary: белый фон, рамка 2px светло-серая, текст тёмный.
- Ghost: без фона, текст серый.
- Gold (VIP): фон — градиент VIP, текст тёмный `#0F0F13`.
**Инпут:** белый фон, рамка 2px `#e5e7eb`, в фокусе рамка `#FF00FF`, радиус 16px, высота 48px, плейсхолдер серый.
**Тумблер (switch):** капсула 48×26px; выкл — серый `#e5e7eb`, кружок слева; вкл — цвет по контексту (фуксия / изумруд для щита / золото для олигарха), кружок справа, плавный переход.
**Пилюля-тег:** капсула высотой 32px, обводка 2px цветом тега (HEX), эмодзи + текст; выбранная — залита цветом тега, текст белый.
**Бейдж:** маленькая капсула, например синий «✓ Verified», изумрудный «✓ 3 общих тега», золотой «👑 VIP».
**Скелетон:** вместо спиннеров — серые `#e5e7eb` плейсхолдеры с лёгкой пульсацией.

# ЧАСТЬ 4. ГЛОБАЛЬНЫЕ КОМПОНЕНТЫ (одинаковы на многих экранах)

## 4.1 Нижний таб-бар (на Ленте, Симпатиях, Чатах, Профиле)
- Высота 64px, прижат к низу, фон — стекло (blur), тонкая верхняя граница.
- 4 пункта слева направо, каждый = иконка (Phosphor) + подпись 10px:
  1. **Лента** — иконка `ph-cards`, подпись «Лента».
  2. **Симпатии** — иконка `ph-heart`, подпись «Симпатии».
  3. **Чаты** — иконка `ph-chat-circle`, подпись «Чаты».
  4. **Профиль** — иконка `ph-user`, подпись «Профиль».
- Активный пункт: иконка и подпись цвета акцента темы (`#FF00FF`), слегка увеличен (scale 1.1).
- Неактивный: серый, opacity 0.5.
- Бейдж-точка (красная, 8px) на иконке при новом событии (например новый лайк на «Симпатии», новое сообщение на «Чаты»).

## 4.2 Модальные окна
- Появляются снизу (bottom-sheet) или по центру, фон затемняется `rgba(0,0,0,0.6)`.
- Скругление верхних углов `2rem`, «ручка» сверху (серая полоска 40×4px).

## 4.3 Тосты/подсказки
- Маленькая капсула, появляется сверху, авто-исчезает. Например «✅ Фото загружено», «⚠️ Фото отклонено модерацией».

## ЭКРАН 1. ОНБОРДИНГ (регистрация) — 9 под-экранов
**Назначение:** первая регистрация. Сейчас это главная боль — выглядит как «анкета госоргана». Нужно сделать ЛЁГКО, ИГРИВО, по ОДНОМУ вопросу на экран.

**Общий каркас всех 9 шагов:**
- Сверху тонкая полоса прогресса (высота 4px), заполняется градиентом бренда по мере шагов.
- По центру вертикально — крупный заголовок-вопрос (H1, font-black) + короткая тёплая подпись серым.
- Контент шага.
- Внизу — кнопка «Далее →» (primary) ИЛИ выбор тапом без кнопки.
- Переход между шагами — плавный слайд вбок (spring).
- Фон — Light-тема (`#FAFAFC` + точки).

**Шаг 1 — Имя.**
Заголовок «Как тебя зовут?», подпись «Только имя — без фамилии 😊». Один большой инпут по центру, плейсхолдер «Имя». Кнопка «Далее →» активна когда введено 2+ буквы.

**Шаг 2 — Возраст. (ВАЖНО, переделать!)**
НЕ календарь, НЕ дата рождения. Заголовок «Сколько тебе лет?». По центру — **большой красивый барабан/слайдер выбора числа от 18 до 60**: огромная цифра возраста в центре (64px font-black, градиент бренда), по бокам соседние числа полупрозрачные, листается свайпом вверх/вниз ИЛИ горизонтальный слайдер-ползунок. Под числом мелким серым: «Тебе должно быть 18 лет или больше». Кнопка «Далее →».

**Шаг 3 — Я (пол).**
Заголовок «Кто ты?». Две большие карточки в ряд (каждая ~48% ширины, высота 120px, радиус 24px): «Парень 👨» и «Девушка 👩». Тап = выбор (рамка фуксия + лёгкая заливка) и сразу переход дальше, без кнопки.

**Шаг 4 — Ищу.**
Заголовок «Кого ищешь?». Три карточки в столбик: «Парня», «Девушку», «Всех». Тап = выбор + переход.

**Шаг 5 — Геолокация.**
Заголовок «Где ты?», подпись «Чтобы показывать людей рядом». Большая primary-кнопка «📍 Найти меня». Ниже — поле поиска города с плейсхолдером «или введи город…»; при вводе появляется выпадающий список городов (Москва, Санкт-Петербург, Новосибирск, Екатеринбург, Казань…), у каждого мелким регион. Выбранный город показывается изумрудной строкой «✓ Москва». Кнопка внизу «Далее →» (или «Пропустить» если ничего не выбрано).

**Шаг 6 — Фото. (сделай САМЫМ красивым шагом — фото это сердце дейтинга)**
Заголовок «Добавь фото», подпись «Первое фото обязательно. До 5 штук». Сетка 5 слотов в виде 3 колонок (аспект 3:4 у каждого):
- Слот 1 крупнее остальных (занимает 2 ячейки по высоте или выделен рамкой), с подписью «Главное».
- Пустой слот: пунктирная рамка `#e5e7eb`, по центру крупный «+».
- Заполненный слот: фото `object-cover`, в углу маленький крестик ✕ на тёмной подложке для удаления.
- Слот в загрузке: лёгкий блюр + крутящийся индикатор фуксией.
Кнопка «Далее →» активна когда есть минимум 1 фото; пока нет — текст кнопки «Добавь хотя бы 1 фото» (disabled).

**Шаг 7 — Интересы (теги).**
Заголовок «Твои интересы», подпись «Выбери до 5». Облако пилюль-тегов (каждый — эмодзи + текст, обводка цветом тега): Спорт 🏋️, Музыка 🎵, Путешествия ✈️, Кино 🎬, Неформалка 💀, Кошки 🐱, Собаки 🐶, Скейт 🛹, Программирование 💻, Готовка 🍳. Выбранный тег заливается своим цветом, текст белый. Сверху счётчик «Выбрано 3/5». При 5 выбранных — остальные слегка тускнеют (нельзя больше). Кнопка «Далее →».

**Шаг 8 — О себе.**
Заголовок «Расскажи о себе», подпись «Необязательно, но +10% к привлекательности 🔥». Textarea (4 строки, радиус 16px), плейсхолдер-пример «Обожаю горы, кофе по утрам и плохие шутки…». Счётчик символов справа внизу «0/150». Кнопка «🚀 В ленту!».

**Шаг 9 — Финал.**
Заголовок «Готово! 🎉», лёгкое конфетти (анимация), подпись «Твоя анкета готова. Время знакомиться!». Большая кнопка «Начать свайпать 💕».

## ЭКРАН 2. ЛЕНТА СВАЙПОВ (главный экран)
**Назначение:** листать анкеты. Стиль — Tinder/Bumble.

**Шапка (сверху, поверх):** слева логотип-текст «CupidBot» фуксией (вес 900). Справа — маленькая кнопка-тумблер «✓ Только Verified» (капсула: выкл — серая обводка, вкл — синяя заливка) и счётчик оставшихся свайпов мелким серым «48 свайпов».

**Карточка анкеты (занимает почти весь экран):**
- Фото человека на весь прямоугольник, скругление 2rem, `object-cover`.
- Сверху на фото — индикаторы фото как в Stories: тонкие палочки по числу фото, активная — белая, остальные полупрозрачные. Тап по левой половине фото — предыдущее фото, по правой — следующее.
- В правом верхнем углу — синий стеклянный бейдж «✓ Verified» (если анкета верифицирована).
- Снизу — затемняющий градиент `from-black/90 to-transparent` на нижнюю половину.
- Поверх градиента снизу слева: **Имя, возраст** (28px, вес 700, белым), под ним строка «📍 Москва · 2 км» (13px, светло-серым), ниже ряд пилюль-тегов (стеклянные, белая обводка) + изумрудный бейдж «✓ 3 общих тега» если есть совпадения.
- При перетаскивании карточки вправо — проявляется зелёный повёрнутый штамп «ЛАЙК ❤️»; влево — красный «НЕТ 👎»; вверх — фиолетовый «СУПЕРЛАЙК ⭐».
- За верхней карточкой видны края 1–2 следующих (стопка).

**Нижняя панель действий (5 круглых кнопок по центру):**
1. ⏪ Rewind — маленькая (44px), белая, иконка amber. (Для Free показывать с подписью-замочком «Premium».)
2. ✕ — большая (56px), белая, иконка тёмная.
3. ♥ — большая (56px), фон градиент бренда, белая иконка, со свечением (цветная тень).
4. ⭐ Суперлайк — маленькая (44px), белая, иконка фиолетовая, с красным бейджем-счётчиком оставшихся суперлайков.

## ЭКРАН 6. ДИАЛОГ (переписка)
**Назначение:** чат с конкретным мэтчем.

**Шапка:** слева «←» назад, рядом маленький аватар + имя (+ синяя ✓) + статус «в сети» (зелёным). Справа: кнопка ⚠️ (пожаловаться, серая) и кнопка ✈️ Telegram. Кнопка Telegram СЕРАЯ/неактивная по умолчанию, под ней мелкая подпись «ещё 4 смс» (счётчик до разблокировки: Free — 15 сообщений, Premium — 5, Kupidon — сразу). Когда условие выполнено — кнопка синеет и активна.

**Лента сообщений:**
- Свои сообщения — справа, фон градиент бренда, белый текст, скругление с острым верхним правым углом (`rounded-tr-none`).
- Чужие — слева, серый фон `#f3f4f6`, тёмный текст, острый верхний левый угол (`rounded-tl-none`).
- **Системная карточка согласия (ConsentBubble)** — по центру, голубая: иконка 🔑, текст «Собеседник хочет в Telegram», две кнопки [Отклонить] (белая) [Одобрить] (сине-индиго градиент).

**Ледоколы** (когда сообщений ещё нет): над полем ввода 3 кнопки-подсказки (стеклянные, фуксия), например «🐾 Собаки или кошки?», «🎵 Какой плейлист?», «✈️ Последняя поездка?». Тап = отправляет как сообщение.

**Поле ввода (снизу):** круглая кнопка-скрепка слева (📎), посередине капсула-инпут «Сообщение…», справа круглая кнопка отправки ➤ (градиент бренда). В режиме 18+ скрепка заменяется на 🔥 (исчезающие фото).

**Исчезающее фото (18+):** входящее фото показывается заблюренным с подписью «🔥 Нажми чтобы посмотреть (1 раз)». После просмотра — «сгорает», вместо него «Фото сгорело 🔥».

# ЧАСТЬ 7. АНИМАЦИИ И ОЩУЩЕНИЯ
- Переходы между шагами онбординга — слайд вбок, spring.
- Свайп карточки — следует за пальцем с лёгким наклоном (rotate), отлетает при отпускании.
- Мэтч — конфетти + «прыжок» модалки (spring).
- Тумблеры — плавный сдвиг кружка.
- Загрузка — скелетоны, НЕ спиннеры (кроме индикатора загрузки фото).
- Тактильная отдача (вибрация) подразумевается на свайпах, лайках, открытии модалок (в дизайне просто учитывай, что это «сочные» моменты).

# ЧАСТЬ 8. ЧЕК-ЛИСТ КАЧЕСТВА (ни одной ошибки)
**Логика:**
- Возраст — ЧИСЛО (слайдер/барабан), НЕ календарь, НЕ дата рождения.
- Цифры лимитов везде консистентны: Free 50 свайпов, Premium 200, Kupidon 500; суперлайки 0/5/5; врывы —/3/15; TG-разблок 15/5/0 сообщений.
- Кнопки ведут в правильные экраны; у каждого экрана проработано пустое состояние и состояние загрузки.
- Фото симпатий — ОТКРЫТЫ (без блюра). Силуэты — только в «кто смотрел» для Free и в VIP-сигнале Олигарха.
**Дизайн:**
- Единые отступы (4/8/12/16/24/32), выравнивание по сетке, единые радиусы и тени.
- Все иконки — одного семейства (Phosphor). Никаких разнородных иконок.
- Контраст текста ≥ AA (читаемо на фото — через градиент-затемнение).
- Все фото/аватары — `object-cover`, без растяжений и искажений. Главное фото всегда есть. Пустые слоты — аккуратные плейсхолдеры с «+», не серые дыры.
- Адаптив от 360px ширины: ничего не обрезается, не наезжает, не вылезает.
- Тексты не обрезаются некрасиво (используй ellipsis там, где нужно).
