3. Механика и Симпатии

Smart Geo (Умная локация)

Если в городе N кончились анкеты, алгоритм берет координаты города, ищет в БД города в радиусе 50-100 км и бесшовно подкидывает их в колоду. Юзер не видит "пустых экранов".

Кнопка ⏪ (Шаг назад)

Отменяет ТОЛЬКО последний свайп влево. Для Free-юзеров кнопка кликабельна, но вызывает поп-ап жесткой монетизации (FOMO): "Случайно упустил судьбу? Верни ее с Premium!". У платников просто возвращает карточку.

Пейволл и Таймер

При исчерпании лимита свайпов экран блюрится. Выводится сочный арт, таймер до 00:00 МСК и переливающаяся кнопка "Стать Купидоном 👑".

4. Чаты и Обоюдное Согласие

Безопасность и Личные границы

Мы не выдаем юзернеймы Telegram просто так. Люди должны захотеть перейти в личку. Механика обмена контактами геймифицирована.

Авто-разблокировка по времени (сообщениям)

Кнопка "Telegram" становится активной сама, если диалог живой:
Free: после 15 сообщений.
Premium: после 5 сообщений.

Мгновенный обмен (Согласие)

Не хотите ждать? Любой юзер может нажать «Запросить ТГ». В чат падает системное сообщение. Если второй нажимает «Одобрить» — защита падает, контакты открыты мгновенно.

6. Режим «Олигарх»

Жесткая монетизация на китах (Whale Monetization). Для тех, кто не светит лицом, но выкупает любые контакты без ожиданий и свайпов.

Глухой Стелс

Анкета полностью удаляется из алгоритма выдачи. VIP становится призраком, которого никто не видит, пока он сам не проявит инициативу.

Золотой Ключ

На карточках появляется кнопка выкупа юзернейма. VIP платит огромную сумму Stars (~$20) — бот мгновенно выдает контакт ТГ в обход всех чатов и мэтчей.

7. Секретная Комната (18+)

9. Архитектура БД (Схема)

Скопируй этот блок для ИИ или бэкендера при проектировании PostgreSQL.

-- Структура ядра (PostgreSQL + SQLAlchemy)

TABLE Users {
  id UUID PRIMARY KEY
  tg_id BIGINT UNIQUE
  name VARCHAR
  gender ENUM('male', 'female')
  search_gender ENUM('male', 'female', 'any')
  tier ENUM('free', 'premium', 'kupidon') DEFAULT 'free'
  swipes_left INT DEFAULT 50
  force_chats_used INT DEFAULT 0 -- Лимит мгновенных чатов (3 для Premium, 15 для Kupidon)
  stars_balance INT DEFAULT 0
  
  -- Флаги безопасности и режимов
  is_18_mode_active BOOLEAN DEFAULT false
  is_stealth_mode BOOLEAN DEFAULT false -- Для Олигархов
  is_shadowbanned BOOLEAN DEFAULT false
}

TABLE Matches {
  id UUID PRIMARY KEY
  user1_id UUID FK(Users)
  user2_id UUID FK(Users)
  is_force_chat BOOLEAN -- Если создан кнопкой "Начать чат" из Симпатий
  tg_unlocked_user1 BOOLEAN -- Дал согласие на ТГ
  tg_unlocked_user2 BOOLEAN -- Дал согласие на ТГ
  messages_count INT DEFAULT 0 -- Таймер до авто-анлока ТГ
}

TABLE Media_Slots {
  id UUID PRIMARY KEY
  user_id UUID FK(Users)
  media_url VARCHAR
  media_type ENUM('photo', 'video') -- Видео занимает 3 слота в UI
  slot_index INT (1-5)
}
